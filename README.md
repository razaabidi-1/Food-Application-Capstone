# Foodie - Expo App (minimal scaffold)

This folder contains a minimal Expo React Native scaffold for the Foodie app required by the final project.

How to run locally:

1. Install dependencies:

```bash
cd repo
npm install
```

2. Start Expo:

```bash
npm start
```

Open in Expo Go or Snack by importing the GitHub repository URL: https://github.com/razaabidi-1/Food-Application-Capstone

Notes:
- This is a minimal scaffold with persistence via AsyncStorage.
- The "Add New Recipe" form accepts an image URL (paste an image link) rather than file uploads.
